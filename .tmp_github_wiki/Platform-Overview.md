# Platform Overview

## What Expl0V1N Solves

Expl0V1N provides a stable, production-oriented security testing platform that combines:
- Vulnerability assessment
- Controlled penetration testing workflows
- Attack surface discovery for bug hunting
- Queue-safe execution and auditability

## Core Principles

- Single target by default for VAPT
- Safe checks before aggressive checks
- Queue-based orchestration
- Resource-aware admission control
- Full traceability and reporting

## Modes

## VAPT Mode

Use for one target (IP, domain, or URL) with a safe scanner sequence.

## Bug Hunting Mode

Use for domain-based recon and attack surface mapping with step-by-step pipeline execution.

## Advanced Hunting

15 optional modules for faster and deeper discovery including ASN mapping, JS secret extraction, takeover detection, API fuzzing, and delta monitoring.

## AI Assistant

In-dashboard assistant for:
- Explanation of findings
- Remediation suggestions
- Prioritization of vulnerabilities
- Executive summaries
